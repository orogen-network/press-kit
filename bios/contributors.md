# Contributors — bios (press template)

This file is the **press-template** schema for contributor bios. The live, public list of named contributors is maintained on the Orogen community page at [orogen.network/community](https://orogen.network/community). Journalists writing about Orogen should pull current names, titles, and quotes from that page; this file is the underlying schema and a place for press-photo references.

---

## Bio schema (one per contributor)

Each contributor entry follows this shape:

### Name, Title

- **Role at Orogen:** one sentence.
- **Prior:** most recent 1–2 roles, employer + tenure + scope (no more than 2 lines).
- **Education / credentials:** terminal degree, certifications relevant to the role.
- **Public profile:** GitHub / LinkedIn / personal site URLs. Verified before publishing.
- **Press photo:** square crop, neutral background, at least 1024 px. Stored at `press-kit/bios/photos/<slug>.jpg`. Cleared for redistribution.
- **Quote (optional):** ≤40 words, on-brief per the Orogen voice guide. Banned-words check applies.
