# Contributors — bios (press template)

This file is the **press-template** schema for contributor bios. Named contributors, titles, quotes, and press photos must be approved before publication; this file is the underlying schema and a place for press-photo references.

---

## Bio schema (one per contributor)

Each contributor entry follows this shape:

### Name, Title

- **Role at Orogen:** one sentence.
- **Prior:** most recent 1–2 roles, employer + tenure + scope (no more than 2 lines).
- **Education / credentials:** terminal degree, certifications relevant to the role.
- **Public profile:** GitHub / LinkedIn / personal site URLs. Verified before publishing.
- **Press photo:** square crop, neutral background, at least 1024 px. Stored at `press-kit/bios/photos/<slug>.jpg`. Cleared for redistribution.
- **Quote (optional):** ≤40 words, on-brief per the Orogen voice guide. Banned-words check applies.

---

## Founding contributors

Named contributor lists are approved per release. Press requesting an interview or an attributed quote should email `press@orogen.network`.

---

## Maintainer credit

Implementation attribution for public press materials rolls to the foundation organization at [github.com/orogen-network](https://github.com/orogen-network). Individual maintainer credit belongs in repository history or approved contributor bios, not reusable boilerplate.
